## How to Edit Wikipedia Without Getting Banned

### DO:
- Create a Wikipedia account (use your real name or a neutral username)
- Start with small, non-controversial edits (fix typos, add citations to existing claims)
- Build edit history for 2-4 weeks before making substantive additions
- ONLY cite peer-reviewed journal articles, not your own site
- Write in neutral, encyclopedic tone (no marketing language)
- Use the Talk page to propose major changes before making them
- Disclose any conflict of interest on your user page

### DON'T:
- Never link to drinkshroome.com from Wikipedia (will be reverted + flagged as spam)
- Never create a page about shroomé or ZSQUARED INC (insufficient notability for a pre-launch brand)
- Never use marketing language or superlatives
- Never cite your own blog posts as sources
- Never make large edits without Talk page discussion
- Never edit from an account that looks like a brand account
